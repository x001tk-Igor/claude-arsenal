# Slice Method — Reference

## Horizontal Slices (broaden reach)

### 1. Regional Slice — Other languages/markets

| Goal | Generic query | Regional slice |
|------|--------------|----------------|
| Find a French haiku publisher | `haiku publisher` | `Moundarren haiku`, `haïku éditeur France` |
| Find Italian espresso machine reviews | `espresso machine review` | `macchina espresso recensione`, `kaffeemaschine test` |
| Research Korean skincare ingredients | `skincare ingredients` | `스킨케어 성분`, `K-beauty ingredient list INCI` |

### 2. Thematic (Authority) Slice — Known institution/expert

| Goal | Generic query | Thematic slice |
|------|--------------|----------------|
| Find haiku publishers worldwide | `haiku publishers` | `Kodansha international haiku`, `Hokuseido Press poetry` |
| Find AI safety research | `AI safety` | `MIRI alignment research`, `Anthropic safety publications` |
| Find indie game dev tools | `game development tools` | `itch.io developer resources`, `Godot engine community` |

## Vertical Slices (dive deeper)

### 3. Specific Term Slice — Niche expert terminology

| Goal | Generic query | Specific-term slice |
|------|--------------|---------------------|
| Deep-dive into haiku | `haiku poetry` | `Matsuo Basho frog poem`, `Kobayashi Issa` |
| Understand coffee processing | `coffee processing methods` | `honey process coffee`, `anaerobic fermentation coffee` |
| Research memory allocation | `memory management programming` | `jemalloc vs tcmalloc`, `slab allocator kernel` |

> **Key rule:** The less common the term, the better the slice. If a term appears on every commercial page — it's useless.

### 4. Syntactic Slice — Alt spellings, scripts, filetypes

| Goal | Generic query | Syntactic slice |
|------|--------------|-----------------|
| French haiku sources | `haiku` | `haïku` (diaeresis filters to French sites) |
| Find Basho in different scripts | `Basho poet` | `松尾芭蕉`, `Bashō`, `Басё` |
| Find specific file types | `machine learning tutorial` | `filetype:pdf "machine learning" syllabus` |
| Find cached/archived pages | `deleted article site.com` | `cache:site.com/article`, `site:web.archive.org "site.com"` |

### 5. Metonymical Slice — Adjacent/related concepts

| Goal | Generic query | Metonymical slice |
|------|--------------|-------------------|
| Explore haiku deeper | `haiku` | `renga collective poetry`, `kigo seasonal reference` |
| Research blockchain | `blockchain technology` | `merkle tree distributed`, `Byzantine fault tolerance` |
| Understand UX design | `UX design principles` | `cognitive load theory interface`, `Fitts's law button size` |

### 6. Peristatic Slice — Quotes, context, fragments

| Goal | Generic query | Peristatic slice |
|------|--------------|------------------|
| Find a half-remembered article | `climate change ocean` | `"the ocean absorbs roughly 30 percent"` |
| Find a specific code error | `React error` | `"Cannot update a component while rendering a different component"` |
| Find a book you vaguely remember | `sci-fi book about memory` | `"total recall" "Philip K. Dick" "We Can Remember It for You Wholesale"` |

## Slice Selection Guide

| Slice | Direction | When to use |
|-------|-----------|-------------|
| Regional | Horizontal | Topic exists in multiple countries |
| Thematic (Authority) | Horizontal | You know a key player in the field |
| Specific Term | Vertical | Need to filter out commercial noise |
| Syntactic | Vertical | Topic has multilingual or technical variants |
| Metonymical | Vertical | Want to explore related knowledge |
| Peristatic | Vertical | Remember fragments or context |

## Combining Slices

Best results come from combining 2-3 slices:
- **Regional + Specific Term** — niche topic in another market
- **Authority + Syntactic (filetype)** — official PDFs from known institutions
- **Metonymical + Peristatic** — adjacent concept with a remembered quote
