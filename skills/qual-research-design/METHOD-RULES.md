# Правила выбора метода качественного исследования

> Дата обновления: 2026-03-20
> Назначение: справочник правил выбора между IDI, ФГ и экспертными интервью
> Описание: дерево решений + факторы для внутреннего скоринга скилла

---

## Три метода

### 1. Глубинное интервью (IDI)
- Индивидуальная беседа, 45–90 минут
- Базовый метод — подходит почти всегда
- Позволяет глубоко проработать личный опыт, мотивацию, барьеры

### 2. Фокус-группа (ФГ)
- Стандартная группа: 5–6 человек, 90–120 минут
- Мини-группа: 3–4 человека — деликатные темы, узкие сегменты
- Диада: 2 человека — когда нужна глубина IDI + минимальная групповая динамика (например, пары «лояльный + нелояльный» для контрастного обсуждения, или когда сложно набрать полную группу)
- Подходит для генерации идей, тестирования концепций, изучения социальных норм
- Групповая динамика — и преимущество, и ограничение

### 2а. Двухуровневая ФГ
- Сначала группы с одной аудиторией (напр. потребители), затем результаты обсуждаются с другой (напр. эксперты, маркетологи клиента, продуктовая команда)
- Когда использовать: нужна реакция «второго уровня» на инсайты — валидация, интерпретация, приоритизация
- Пример: ФГ с покупателями → результаты обсуждаются с бренд-командой клиента

### 3. Экспертное интервью
- Индивидуальная беседа с труднодоступным респондентом, 30–60 минут
- Другой стиль модерации: уважение к экспертизе, быстрый переход к сути
- Высокие инсентивы, сложный рекрут

## Дерево решений

```
Тема сенситивная?
├── ДА → только IDI (нельзя ФГ)
│   └── Аудитория премиум/люкс/C-level/узкие специалисты?
│       ├── ДА → экспертные интервью
│       └── НЕТ → стандартные IDI
│
└── НЕТ → проверить дальше
    │
    ├── Нужен индивидуальный путь (CJM, journey)?
    │   └── ДА → только IDI (групповая дискуссия не даст последовательность)
    │
    ├── Аудитория премиум/люкс/C-level/узкие специалисты?
    │   └── ДА → экспертные интервью
    │
    ├── Нужна групповая динамика?
    │   ├── ДА (тест концепций, генерация идей, соц. нормы) → ФГ
    │   └── НЕТ → IDI
    │
    └── Аудитория однородная внутри сегмента?
        ├── ДА → ФГ возможна
        └── НЕТ → IDI (разнородные профили нужно прорабатывать индивидуально)
```

## Когда НЕЛЬЗЯ фокус-группы

| Ситуация | Почему |
|---|---|
| **Сенситивная тема** (здоровье, финансовые проблемы, личная гигиена, стигматизированное поведение) | Социальная желательность: люди в группе дают «правильные» ответы, замалчивают реальный опыт |
| **Построение CJM / customer journey** | Путь клиента — индивидуальная последовательность. Группа обсуждает «снимки», а не нарратив. Теряются триггеры, эмоциональные пики, точки решений |
| **Конфиденциальные темы в B2B** | Участники из одной индустрии боятся раскрыть коммерческую информацию при конкурентах |
| **Очень неоднородная аудитория в группе** | Доминантные участники подавляют мнения, конформизм, потеря индивидуальных инсайтов |

---

## Когда подходят фокус-группы

| Ситуация | Почему |
|---|---|
| **Тестирование концепций / упаковки / коммуникаций** | Групповая реакция показывает, как сообщение воспринимается в социальном контексте |
| **Генерация идей, брейнсторм** | Участники дополняют и развивают идеи друг друга |
| **Изучение социальных норм и категорийных фреймов** | Группа показывает, как люди обсуждают категорию между собой |
| **Быстрое покрытие** | 4 группы × 5–6 человек = 20–24 мнения за 2 дня |

**Важно:** для сенситивных тем и построения CJM — только глубинные интервью (см. «Когда НЕЛЬЗЯ фокус-группы»).

---

## Правила состава групп: кого НЕ смешивать

В одной фокус-группе **НЕ должны быть вместе:**

| Кого разделять | Почему |
|---|---|
| **Разный статус / власть** (начальники ≠ подчинённые) | Подчинённые замолкают или подстраиваются |
| **Разные уровни иерархии** | Старшие доминируют, младшие конформны |
| **Эксперты и новички** | Эксперт давит авторитетом, новичок не высказывается |
| **Разные возрастные группы подростков** (12-летние ≠ 16-летние) | Совершенно разный опыт и язык |
| **Лояльные и нелояльные** в одной группе (при бренд-стратегии) | Лояльные защищают бренд, нелояльные замалчивают критику или наоборот — дискуссия уходит в конфликт |

**Принцип:** группа должна быть **гомогенной по ключевому признаку**. Гомогенность определяется опытом и ролью, а не только демографией.

## Когда экспертные интервью

| Аудитория | Признаки |
|---|---|
| **Люкс / премиум-потребители** | Высокий доход, не ходят на обычные исследования, требуют особого подхода |
| **C-level / топ-менеджмент** | Плотный график, высокие инсентивы, нужен модератор с отраслевой экспертизой |
| **Узкие специалисты** | Врачи, архитекторы, шеф-повара — маленький пул, сложный рекрут |
| **Сложный B2B** | Закупщики enterprise-решений, IT-директора — решения принимаются долго и коллегиально |

Особенности модерации экспертных:
- Не объяснять очевидное — уважать экспертизу респондента
- Быстрый переход к стратегической глубине
- Меньше проективных техник, больше прямых вопросов

## Правила расчёта количества

### IDI

| Тип сегмента | Количество | Комментарий |
|---|---|---|
| Приоритетный, гомогенный | 5–7 | Насыщение по основным темам |
| Приоритетный, неоднородный | 7–9 | Широкий разброс внутри сегмента |
| Второстепенный | 2–3 | Обнаружение паттернов, не полное насыщение |
| Минимум при жёстком бюджете | 4 | Порог, ниже которого данные ненадёжны |

### ФГ

| Контекст | Количество групп на сегмент | Формат | Комментарий |
|---|---|---|---|
| **Маркетинговое исследование** (отдельный проект, нормальный бюджет) | от 3 | Стандартная (5–6 чел.) | Полноценное насыщение |
| **Бренд-стратегия / ребрендинг / бренд-платформа** | 2 | Стандартная (5–6 чел.) | Допустимо меньше — исследование входит в более широкий проект |
| Несколько городов | Мин. 1 на город | Стандартная (5–6 чел.) | Региональные различия |
| Узкий сегмент, сложный рекрут | 2 | Мини-группа (3–4 чел.) | Когда трудно набрать полную группу |
| Контрастное обсуждение, пары | по задаче | Диада (2 чел.) | Глубина IDI + минимальная групповая динамика |
| Минимум | 2 | 5 чел. | Порог надёжности |

### Экспертные интервью

| Тип сегмента | Количество | Комментарий |
|---|---|---|
| Основной | 3–5 | Достаточно при качественном рекруте |
| Проверочный | 2 | Минимум для кросс-проверки |

### Логика при бюджетных ограничениях

Порядок урезания (от менее болезненного к более):
1. Сократить второстепенные сегменты до 1–2 интервью
2. Убрать проверочные сегменты
3. Уменьшить основные сегменты до 4 (но предупредить о рисках)
4. Объединить похожие подсегменты

## Оси сегментации для исследований бренд-стратегии

### Лояльность (когда включать — см. SKILL.md, шаг 2в)

| Тип | Описание | Зачем в исследовании |
|---|---|---|
| **Лояльные** | Регулярно покупают бренд клиента | Понять драйверы лояльности, что удерживает |
| **Нелояльные / покупатели конкурентов** | Покупают в категории, но не бренд клиента | Понять барьеры, что не так с брендом, триггеры переключения |
| **Свитчеры** | Чередуют бренды | Понять критерии выбора в моменте |

### Демография

Обычные оси для комбинирования с лояльностью:
- **Пол** — если продукт имеет гендерную специфику
- **Возраст** — обычно 2–3 возрастные когорты (напр. 25–35 / 35+)
- **Доход** — если ценовой сегмент продукта критичен

### Принципы комбинирования

- Не множить оси без необходимости: каждая дополнительная ось × 2 удваивает количество интервью
- Если оси дают >15 подсегментов — упрощать: объединять второстепенные, сокращать до паттерн-проверки
- Основная ось (обычно лояльность или ключевая демография) получает больше интервью
- Второстепенные оси — минимальное покрытие для сравнения

## Факторы сенситивности темы

### Высокая сенситивность (→ только IDI)
- Здоровье, особенно стигматизированные состояния
- Финансовые проблемы, долги
- Личная гигиена, интимная сфера
- Зависимости (алкоголь, азартные игры)
- Социально неодобряемое поведение
- Детское питание, воспитание (страх осуждения)

### Средняя сенситивность (→ IDI предпочтительнее)
- Доход и траты (люди занижают/завышают в группе)
- Причины выбора «дешёвого» варианта (стыд)
- Отношение к люксу (хвастовство или стеснение)
- Проблемы с внешностью

### Низкая сенситивность (→ ФГ возможна)
- Выбор продуктов питания, бытовой химии
- Предпочтения в одежде (масс-маркет)
- Медиапотребление
- Путешествия, хобби

## Стратегии формирования выборки

Три стратегии, применимые в агентской практике. Выбор зависит от задачи исследования.

### 1. Максимальной вариации
**Суть:** покрыть максимально разные позиции — паттерны, которые устойчивы при разнообразии, самые ценные.
**Когда:** бренд-стратегия (лояльные + нелояльные + свитчеры), исследование категории, поиск инсайтов для позиционирования.
**Пример:** исследование для ребрендинга сети кофеен → берём постоянных гостей, тех кто ходит к конкурентам, и тех кто чередует. Разный возраст, разные районы. Ищем, что объединяет всех — и что разделяет.

### 2. Гомогенная
**Суть:** описать одну подгруппу максимально глубоко.
**Когда:** один сегмент нужно понять детально — драйверы, барьеры, путь клиента.
**Пример:** 5–7 IDI только с лояльными покупательницами 25–35 лет — понять, почему именно они остаются с брендом.

### 3. Цепная (снежный ком)
**Суть:** каждый респондент рекомендует следующего. «Кто знает больше всего о…?»
**Когда:** нет базы для рекрута, узкая аудитория.
**Пример:** B2B-исследование закупщиков промышленного оборудования — первый контакт через клиента, дальше по рекомендациям.
**Ограничение:** использовать только в крайних случаях. Агентству нужно продавать клиенту фиксированное количество интервью — при снежном коме сложно гарантировать точное число и сроки. Предлагать только когда стандартный рекрут невозможен, и обязательно предупреждать клиента о рисках по срокам.
